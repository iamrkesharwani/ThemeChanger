# Theme Switcher

A lightweight theme switcher with 6 color schemes, keyboard shortcuts, and persistent preferences.

## Features

- 6 built-in themes: Light, Dark, Nature, Sunset, Ocean, and Purple
- Quick toggle button for light/dark mode switching
- Keyboard shortcuts (1-6 for themes, T for dark mode)
- Toast notifications on theme change
- Persistent theme storage across sessions
- Active theme indication

## Technology Stack

- Vanilla JavaScript (ES6+)
- CSS3 with custom properties
- LocalStorage API

## Installation

1. Include the JavaScript file in your HTML
2. Add theme buttons with `data-theme` attributes
3. Open in your browser

## Usage

### Switch Themes
- Click theme buttons
- Press keys 1-6 for instant theme switching
- Press T to toggle dark mode
- Use the quick toggle switch

### Keyboard Shortcuts
- `1-6` - Switch to specific themes
- `T` - Toggle dark theme

## Customization

Add new themes by updating `THEME_CLASS_MAP` and defining corresponding CSS classes with custom properties.

## Browser Compatibility

Requires modern browser with ES6 support, CSS variables, and localStorage.
